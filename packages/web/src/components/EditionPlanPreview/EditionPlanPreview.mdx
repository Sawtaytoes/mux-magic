# EditionPlanPreview

Preview component for the edition-folder organization plan. The NSF
(`nameSpecialFeaturesDvdCompareTmdb`) pipeline emits an `editionPlan`
event **before** any moves happen, summarizing every planned file move.
This component renders that preview so users can see what is about to
occur before the moves execute.
